# Just Ask God Ministries App

A Christian community app designed for Bible study, prayer, worship, fellowship, and resource support.

## Features
- Bible Study Hub
- Prayer Room (requests + answered prayers)
- Worship Audio & Spoken Word
- Community Groups & Conversations
- Resource & Referral Finder
- Testimony Sharing
- Notifications for daily scripture & prayer

## Tech Stack
- React Native (mobile)
- Node.js / Express (backend)
- MongoDB or Firebase (database)
- GitHub Actions (CI/CD)

## Vision
To create a digital sanctuary where people can grow spiritually, find support, and stay connected to the message of Jesus Christ.
