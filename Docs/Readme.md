# Just Ask God Ministries App

A Christian community app designed for Bible study, prayer, worship, fellowship, and resource support.

## Core Features
- Bible Study Hub (multiple translations, notes, highlights)
- Prayer Room (submit, view, and mark answered prayers)
- Worship & Spoken Word Audio
- Community Groups & Conversations
- Resource & Referral Finder
- Testimony Sharing
- Daily Scripture & Prayer Notifications

## Tech Stack
- React Native (mobile app)
- Node.js + Express (backend API)
- MongoDB or Firebase (database)
- GitHub Actions (CI/CD)

## Vision
To create a digital sanctuary where believers can grow spiritually, find support, and stay connected to the message of Jesus Christ.

## Status
This repository contains the starter code for both the mobile app and backend server.
